Convolutional and fully-connected neural networks C++ library

What is nnForge?

nnForge is a framework for training convolutional and fully-connected neural networks. It includes CPU and GPU (CUDA) backends.

Layers

The framework has a number of layers defined:

  • Convolutional (1D, 2D, 3D, and 4D)
  • Sparse (in feature map dimension) convolutional (1D, 2D, 3D, and 4D)
  • Fully connected
  • Local contrast subtractive (CPU backend - no limitations, GPU backend - 2D only)
  • RGB->YUV conversion (CPU backend: forward propagation only, GPU backend - forward + backward prop)
  • Max spatial subsampling (1D, 2D, 3D, and 4D)
  • Average spatial subsampling (1D, 2D, 3D, and 4D)
  • Maxout - arXiv:1302.4389
  • Rectification - |x|
  • Hyperbolic tangent - 1.7159*tanh(0.666666*x)
  • Sigmoid
  • ReLU (rectified linear unit) - max(0,x)
  • Parametric ReLU - arXiv:1502.01852
  • Softmax
  • Dropout

Training

The library implements Stochastic Gradient Descent training algorithm.

Mini-batch training supported.

Regularization

  • Weight decay

Error functions

Error functions available:

  • MSE - Mean Squared Error
  • SHL - Squared Hinge Loss
  • NLL - Negative Log-Likelihood
  • CE - Cross-Entropy

Download

The package contains nnForge framework as well as examples - applications using the framework.

Prerequisites

The framework depends on Boost and OpenCV.

If you want to use CUDA backend you will also need CUDA Toolkit installed and cuDNN v2 installed.
